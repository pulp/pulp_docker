``pulp_docker`` Plugin
======================

This is the [pulp_docker](https://docs.pulpproject.org/plugins/pulp_docker/index.html)  Plugin for [Pulp 2 Project](https://docs.pulpproject.org/).
This plugin provides Pulp 2 with support for docker images.
